• HTML to define the content of web pages
  • CSS to specify the layout of web pages
  • JavaScript to program the behavior of web pages

The DOCTYPE html declaration represents the document type, and helps browsers to display web pages correctly. Attributes provide additional info about elements. Current version – HTML 5 released in 2014

New HTML5 elements are:

  • New semantic elements like header, footer, article, and section.
  • New attributes of form elements like number, date, time, calendar, and range.
  • New graphic elements: svg and canvas.

New multimedia elements: audio and video.

New HTML5 API's (Application Programming Interfaces):
  • HTML Geolocation
  • HTML Drag and Drop
  • HTML Local Storage
  • HTML Application Cache
  • HTML Web Workers
  • HTML SSE
HTML Charset: ASCII - American Standard Code for Information Interchange ANSI – American National Standard Institute UTF-8 (Unicode Transformation Format) covers almost all the characters and symbols in the world. o UTF-8 is identical to ASCII for the values from 0 to 127. o UTF-8 does not use the values from 128 to 159. o UTF-8 is identical to both ANSI and 8859-1 for the values from 160 to 255. o UTF-8 continues from the value 256 with more than 10 000 different characters.
